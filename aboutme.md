# 🦆 Duck Of Apocalypse
> *Automating the bridge between thought and AI.*

## 🎯 Смысл проекта (The "Why")
Разработка с использованием ИИ-агентов требует постоянного переключения контекста. Каждый раз нужно:
1. Открыть терминал.
2. Перейти в нужную папку проекта.
3. Запустить команду агента (например, `gemini` или `claude`).
4. Включить диктовку, чтобы начать говорить.

Это занимает 5-10 секунд и сбивает поток мыслей ("Flow"). **Duck Of Apocalypse** убирает это трение.

Он превращает всю эту рутину в **одно нажатие клавиши**.

## 🚀 Что он делает (The Workflow)
При нажатии глобального хоткея (например, `Cmd+Key` через Hammerspoon):

1. **Мгновенный запуск**: Duck просыпается (без тяжелого GUI).
2. **Контекст**: Открывает твой любимый терминал (Ghostty, iTerm, Terminal).
3. **Навигация**: Автоматически переходит в папку с проектами (`~/aiwork`).
4. **Агент**: Запускает выбранного ИИ-собеседника.
5. **Голос**: (*Главная фишка*) Сама нажимает кнопку диктовки (F5), чтобы ты мог сразу говорить.

Ты нажимаешь одну кнопку и сразу начинаешь диктовать задачу.

## 🛠 Технологии под капотом
Проект построен как "швейцарский нож" для автоматизации:

*   **Core**: Python (Typer для CLI, Rich для красивого вывода).
*   **Automation**:
    *   **macOS**: Native AppleScript (без прав Accessibility для keystrokes, используем `do script`).
    *   **Linux**: `xdotool`, `wmctrl` и нативные команды терминалов.
*   **Deployment**: Сборка в один бинарный файл через PyInstaller (работает без установки Python).
*   **Integration**: Глубокая интеграция с **Hammerspoon** для работы на уровне системы.

## 🔮 Философия
**"Duck"** — это ссылка на метод "Rubber Duck Debugging" (метод утенка), только эта утка:
1. Вооружена инструментами апокалипсиса (AI).
2. Полностью автоматизирована.
3. Всегда готова слушать.

Это не просто скрипт, это экзоскелет для твоего продуктивного процесса.
